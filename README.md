# forecast-reporter
A custom Mocha.js reporter optimized for long-running time asynchronous tests.

## Features
* Test title first, result second
* Declares total number of tests when start

## Usage
1. Clone this repo to local
2. Move `forecast.js` to your project folder
3. Launch Mocha with `"mocha -t [insert timeout] --reporter /path/to/forecast.js"`
